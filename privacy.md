# Privacy Policy — Pantry Whisperer

_Last updated: July 2026_

Pantry Whisperer is a personal software project built and used solely by its developer. It is not offered as a public product or service, and it is not intended for use by anyone other than the developer.

## What this application does

Pantry Whisperer connects to the developer's own Pinterest account via Pinterest's official OAuth authorization flow. It reads the developer's own boards and pins in order to extract recipe information (ingredients, steps, and related details) for personal meal planning and grocery list generation.

## What data is accessed

- Pinterest board and pin data belonging solely to the developer's own account, accessed via the Pinterest API (`pins:read`, `boards:read` scopes).
- No other Pinterest users' data is accessed, collected, or processed.

## How data is used and stored

- Data retrieved from Pinterest is used only to build the developer's personal recipe database and related meal-planning features.
- Data is stored privately, for the developer's own use only.
- No data is sold, shared, rented, or otherwise disclosed to any third party.
- No data is used for advertising or marketing purposes.

## Third-party services

This application may use third-party AI services (such as Anthropic's Claude API) to process recipe text extracted from web pages linked by the developer's own Pinterest pins. No Pinterest user data beyond the developer's own account is transmitted to these services.

## Contact

This is a personal, non-commercial project. Questions can be directed to the developer via the contact information listed on the project's GitHub repository.
